## 遊戲特色

### 🔢 數獨遊戲特色
- **三種難度等級**：簡單（40個空格）、中等（50個空格）、困難（55個空格）
- **智能數獨生成**：每次遊戲都會產生有唯一解的全新數獨謎題
- **即時驗證**：自動檢測數字衝突並以紅色標示錯誤
- **計時功能**：記錄遊戲時間，支援暫停/繼續
- **提示系統**：卡住時可獲得智能提示
- **完整驗證**：一鍵檢查整個棋盤的錯誤

### 🐱🐶 井字遊戲特色
- **雙人對戰模式**：和朋友一起玩的經典模式
- **電腦對戰模式**：挑戰智能 AI 對手
- **可愛主題設計**：貓咪和狗狗的萌系介面
- **流暢動畫效果**：每個動作都有精美的視覺回饋
- **智能 AI**：電腦對手具備策略思維，提供真正的挑戰

### 共同特色
- **現代化設計**：美觀的漸層背景和流暢動畫效果
- **響應式設計**：完美支援桌面和行動裝置
- **直觀操作**：簡單易懂的使用介面
- **無廣告**：純淨的遊戲體驗
- **離線可用**：無需網路連線即可遊玩

## 如何遊玩

### 🔢 數獨遊戲
**基本規則**
- 在 9×9 的格子中填入數字 1-9
- 每一行都包含 1-9 各一次
- 每一列都包含 1-9 各一次  
- 每個 3×3 的小區塊都包含 1-9 各一次

**操作說明**
1. 選擇難度等級開始遊戲
2. 點擊空格，再點擊數字按鈕填入
3. 使用提示、驗證和暫停功能
4. 完成所有格子即可獲勝

### 🐱🐶 井字遊戲
**基本規則**
- 在 3×3 的格子中輪流放置符號
- 率先讓三個符號連成一線者獲勝
- 連線可以是橫向、縱向或對角線

**操作說明**
1. 選擇雙人對戰或電腦對戰模式
2. 點擊空格放置你的符號（貓咪）
3. 嘗試搶先連成三個一線
4. 阻止對手連線並爭取勝利

## 技術規格

### 開發技術
- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **設計**：響應式設計、CSS Grid、Flexbox
- **演算法**：回溯法 (Backtracking) 數獨求解和生成

### 檔案結構
```
game-collection/
├── index.html              # 遊戲選擇主頁面
├── sudoku/                 # 數獨遊戲資料夾
│   ├── index.html          # 數獨遊戲主頁
│   ├── style.css           # 數獨遊戲樣式
│   └── script.js           # 數獨遊戲邏輯
├── tic-tac-toe/            # 井字遊戲資料夾
│   └── index.html          # 井字遊戲（內含樣式和腳本）
└── README.md               # 專案說明文件
```

### 核心功能實作
**數獨遊戲**
- **數獨生成器**：使用回溯演算法生成完整解答，再隨機移除數字
- **衝突檢測**：即時檢查行、列和 3×3 區塊中的重複數字
- **遊戲狀態管理**：處理計時器、暫停、重置等功能

**井字遊戲**
- **智能 AI**：使用 Minimax 策略提供有挑戰性的電腦對手
- **遊戲邏輯**：自動檢測勝利條件和平手狀態
- **模式切換**：支援雙人和單人模式無縫切換

**共同功能**
- **UI 互動**：流暢的視覺回饋和動畫效果
- **響應式設計**：適配各種裝置螢幕尺寸
